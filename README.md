#### 常见问题

1、mac安装后打开提示已损坏？

<div>
  <img width="250" src="./img/已损坏提示.jpeg">
</div>

系统的“隐私与安全性”中允许“任何来源”

终端输入命令：

```
sudo xattr -d com.apple.quarantine /Applications/wpspaster.app
```
